# Deepatkar Patil — Portfolio Website

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a dark, high-contrast design with red accents, smooth animations, and a fully responsive layout.

## Features

- **Sticky navigation** with scroll progress bar and mobile hamburger menu
- **Hero section** with animated ring decorations and key stats
- **About section** with skills grid
- **Projects section** with hover-animated cards
- **Resume section** with timeline for education/experience and skills tags
- **Achievements section** with icon cards
- **Contact section** with links and a message form
- **Custom cursor dot** and smooth scroll behavior
- **Scroll reveal animations** on all sections

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts: Bebas Neue, DM Sans, JetBrains Mono

## Getting Started

Just open `index.html` in any modern browser — no build step or dependencies required.

## Deployment

Hosted on **GitHub Pages**.

1. Push your code to a GitHub repository
2. Go to **Settings → Pages**
3. Under *Source*, select the branch (e.g. `main`) and set the folder to `/ (root)`
4. Save — your site will be live at `https://<your-username>.github.io/<repo-name>/`

## Customization

- Colors are defined as CSS variables in `:root` — change `--red` to restyle the entire accent theme
- Update the hero stats, project cards, timeline items, and contact links directly in the HTML
- Swap the emoji icons in `.card-img` and `.ach-icon` for images or SVGs as needed
