My Projects

Seller-Buyer Management System
Simulates real-world seller and buyer interaction. Sellers list and manage products; buyers can browse, bargain, and purchase. Includes full inventory management system.
🚌
Smart Transportation Advisory
Recommends the best transportation mode based on live conditions. Displays AQI, weather, humidity, and temperature. M-Indicator inspired design with real-time travel suggestions.
🔬
Skin Cancer Detection System
ML-powered web app that detects skin cancer from uploaded images. Provides classification with confidence score output and real-time result display. Built at API Mania Hackathon.

4-bit Password Lock System
Digital logic-based security system using combinational circuits. Implements AND/NOT gate logic using 74HC08 and 74HC04 ICs. Prototyped on breadboard in Tinkercad.

Mechanical and electronics project simulating human hand movement using 4 degrees of freedom. Integrates servo motors with microcontroller for precise gesture control.
